Low Rate (LRT - 1 sps) Navigation, State Parameter, and Microphysics
Flight-Level Data
https://data.eol.ucar.edu/dataset/605.004

Summary
This dataset includes airborne measurements obtained from the NSF/NCAR HIAPER
GV aircraft (Tail Number N677F) during the SPICULE project. This dataset
contains low rate navigation, state parameter, and microphysics flight-level
data in netCDF format.

Versions
  0.1
  0.9: Last preliminary version processed with all housekeeping variables
included. If you would like access to this version of the dataset,
please contact the SPICULE Project Manager Cory Wolff.
  1.0
  2.0: Updates the CDP binning to 3 separate calibrations over the course of
the project and removes the 0th order bin.
  2.1: Corrects the assignment of EWX to the left dewpointer. Corrects
inclusion of merged data values for CVI and VCSEL variables, which only
impacts users who downloaded the Version 2.0 dataset. Version 1.0
contained the correct merged CVI and VCSEL variables.
